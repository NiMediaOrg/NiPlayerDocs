# 安装

# 项目内引入

请选择你喜欢的包管理器进行安装...

# npm

npm i niplayer

# yarn

yarn add niplayer

# pnpm

pnpm i niplayer

# CDN网页端体验

如果你只是想通过html的script标签来引入从而单纯在网页端体验的话，建议通过CDN导入最新版本的umd格式的JS文件.

<script src="https://cdn.jsdelivr.net/npm/niplayer@latest/dist/player.umd.js"></script>

导入完成之后，接着在你创建的html文件中的script标签内如此code，

WARNING

CDN导入的话，NiPlayer中一切导出的组件或者对象都绑定在Player下面，只能通过Player.xxx这种形式访问

<div id="video">
    <!-- <video playsinline="true" x5-video-player-type = "true" src="https://novaex.cc/monster.mp4"></video> -->
</div>
<script>
    let video = document.getElementsByTagName("video")[0]
    let player = new Player.default({url:"https://novaex.cc/monster.mp4" , container: document.querySelector("#video"),streamPlay:true});
</script>
